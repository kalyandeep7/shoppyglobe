# ShoppyGlobe - E-commerce Application

A React e-commerce app built with Vite, Redux Toolkit, and React Router.

## GitHub Repository
https://github.com/YOURUSERNAME/shoppyglobe

## How to Run
npm install
npm run dev

## Features
- Product listing with search
- Product detail page
- Cart with quantity controls
- Checkout with order placement
- Redux state management
- Lazy loading & code splitting
- Responsive design
